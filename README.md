# スマスロ目標 — 個人アーカイブ

「たかどらのスロ塾」(suroschool.jp) の会員コンテンツを**自分専用に保存・閲覧**するための個人サイト。
正規会員＋サイト代表の許可のもと、自分用メモとして構築（[再配布・公開禁止](#-注意)）。

## 何ができる
- **機種ごとの狙い目・解析情報・集計値**をオフラインで高速閲覧（スロット店内の電波が悪い環境向け）
- 機種一覧の**検索 / 並び替え / 解析ありフィルタ / お気に入り★**
- **期待値計算ツール**（機械割ベース収支・期待差枚→時給・天井期待値）
- お知らせ・資料ページ

## 収録データ（2026-06-16 時点）
- 機種: 160（解析情報あり 130 / 集計値ダッシュボード 2）
- お知らせ: 258 / 固定ページ: 19
- 画像: ローカル保存済み（オフライン表示）

## 使い方（PC）
`site/index.html` をブラウザで開くだけ。

## 使い方（iPhone・店内オフライン）
1. iPhone でサイトを一度開く（下記の配信方法のいずれか）
2. Safari の共有 → **「ホーム画面に追加」**
3. 以降はホーム画面アイコンから起動 → **電波が無くても全データを即表示**（Service Worker がキャッシュ済み）

### 配信方法
- **おすすめ（完全プライベート）**: 自宅PCで `cd site && python3 -m http.server 8080` → 同じWi‑Fiの iPhone から `http://<PCのIP>:8080/` を開いて「ホーム画面に追加」。一度キャッシュすれば店内オフラインでOK。
- private GitHub リポジトリにコードを保管。閲覧は上記ローカル配信 or 任意の非公開ホスティングで。

## 再ビルド / 更新（新台追加時など）
```bash
cd scraper
node login.mjs   # 初回 / セッション切れ時のみ（ブラウザでDiscordログイン）
node crawl.mjs   # 全ページ取得（低負荷・直列・キャッシュskip・再開可能）
node parse.mjs   # HTML → JSON
node dash.mjs    # 集計値ダッシュボード取得
node mirror.mjs  # 画像ローカル化
node gen.mjs     # サイト + PWA 生成
```

## 構成
- `scraper/` … 取得・変換・生成スクリプト
- `data/json/` … 構造化データ / `data/raw/` … 生HTML（gitignore）
- `site/` … 生成された個人サイト（PWA）

## ⚠️ 注意
- 本サイトは**有料会員コンテンツ**を含む。リポジトリは **private 必須**、**絶対に公開しない**こと。
- セッション情報（`scraper/.profile/`, `data/auth.json`）はコミットしない（.gitignore 済み）。
